## What is Painel Wallhack?

Painel Wallhack is a simple overlay-based utility that exposes enemy positions and nearby objects through solid surfaces in compatible games. Its panel-driven layout keeps controls close at hand, so visibility options can be switched without breaking focus in the middle of play. For players who want better awareness of threats hidden by walls or other obstacles, it offers a direct and practical workflow.

The application is designed for Windows and does not require complicated setup before use. Because it is built to stay light on system resources, it aims to keep CPU and GPU overhead low while still giving fast access to the available options. From training map awareness to handling high-tempo matches, Painel Wallhack is meant to provide a clear visual advantage.

## Configuration

Settings are saved in a `config.ini` file located beside the executable. If you prefer to edit values manually, you can open the file in any text editor and change the defaults:

```ini
[Display]
opacity=0.7
color=#FF0000
show_distance=true

[Controls]
toggle_key=F1
menu_key=F2
```

You can also manage every option through the in-game panel, and any changes will be written automatically when the app exits.
